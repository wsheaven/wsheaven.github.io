WDD 230 Lesson 4 Class Activity

This exercise we are going to use CSS Grid to copy a wireframe provided. After you will make another file are rewrite that code using CSS Flexbox. Once finished with the activity discuss the advantages and disadvantages of using each CSS Grid and Flexbox with others around you or on the discussion board.

If you haven't completed W04 Activity: CSS Layouts review it quickly before starting. If you think you need more practice you can do the CSS Grid Garden or Flexbox Froggy activity today to practice and work on this activity on Friday.

Instructions

  1. Do a basic html outline for this page based on the wireframe.
  2. Use CSS Grid to style your html to look like the wireframe.
  3. Once you have the desired css file either comment out the grid or make a new file to hold you flexbox css.
  4. Use CSS Flexbox to style your html to look like the wireframe.

Hints, Tips and Extras

  • Make sure you DON'T delete your css. You will be comparing it later.
  • What does the CSS gap Property do?
  • What does flex grow and flex shrink do?
  • How would you move the cards from bottom to top in Grid?
  • How would you move the cards from bottom to top in Flexbox?
